那些servlet的基本知识你掌握了吗？

  • java servlet

一、什么是servlet

servlet是一种独立于平台和协议的服务器端java应用程序。通过servlet可以生成动态的web页面。
同时，使用servlet还可以在服务器端对客户端的请求进行处理，控制程序的执行。

二、servlet的作用

交互式的浏览和更新数据，并生成动态的页面内容展示。

三、servlet的生命周期

1.加载和实例化--servlet容器创建servlet实例
2.初始化--调用init()方法
3.提供服务--调用service()方法
4.销毁--调用destroy()方法
